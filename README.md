Road Accident Prediction Model

Description
This project focuses on predicting road accidents using machine learning techniques.  
The model analyzes various factors such as weather conditions, road type, time, and other features to estimate the likelihood of accidents.

Tools & Technologies Used
- Python 
- Jupyter Notebook  
- Scikit-learn  
- Pandas  
- NumPy  
- Matplotlib / Seaborn 
- ML Algorithms


📊 Dataset
The dataset contains information about road accidents including features like:
- Weather conditions  
- Road type  
- Time of accident  
- Location details  
- Vehicle information  

Dataset : Available in data/RTA Dataset.csv
